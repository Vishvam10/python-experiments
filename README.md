## Python Experiments

A sandbox repository for exploring Python internals, performance, and execution behavior.

Currently, the repository contains experiments around Python execution models, including threads, processes, and subinterpreters. Each experiment is self-contained, with its own benchmarks, plots, and analysis.

### Experiments

| Experiment       | Description                                                                                      | Link                                                     |
| ---------------- | ------------------------------------------------------------------------------------------------ | -------------------------------------------------------- |
| Execution Models | Benchmark threads, processes, and subinterpreters across CPU, IO, memory, and network workloads. | [execution-models/README.md](execution-models/README.md) |


### Getting Started

Navigate into any experiment directory (e.g., `execution-models/`) and follow its README for setup, running benchmarks, and viewing results.
